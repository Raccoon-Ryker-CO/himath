Operations

Plus

Plus is adding numbers together so for example:
1 + 1 is equal to 2.

Minus

Minus is removing numbers so think of 2 - 1 being 1.
That is because we are removing 1 from 2.

Multiplication

Things are gonna get tricky!

Multiplication is all about skip-counting so if we did 5 * 5,
We would need to skip-count by 5's 5 times like this: 5, 10, 15, 20, 25. So the answer is 25.

Division

If you know multiplication, Division is the oppisite of multiplication so
if we had a multiplication question and we had to convert it into division,
we need to put the answer of the multiplication question into the first number of the division question and the first
number of the multiplication question is the answer of the division question so 25 / 5 would equal 5 because 5 * 5 equals 25.
